HTML

Hypertext Markup Language(HTML) is the standard markup languge for creating web pages and web applications. Web browsers receive HTML documents from a web server of from local storage and render them into multimedia web pages. HTML describes the structure of a web page semanticaly and originally included cues for the appearance of the document.

하이퍼텍스트 마크업 언어(Hypertext Markup Language, HTML은 웹 페이지와 웹 응용 프로그램을 만드는 표준 마크업 언어입니다. 웹 브라우저는 로컬 저장소에서의 웹 서버에서 HTML 문서를 수신하여 멀티미디어 웹 페이지로 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 설명하고 원래 포함된 문서 모양을 위한 단서들을 설명합니다.
