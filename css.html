CSS

Traditional HTML has many limitations in designing and changing web documents from time to time. To complement this, style sheets are designed and the standard for style sheets is CSS. It is also simply referred to as a style sheet.

기존의 HTML은 웹 문서를 다양하게 설계하고 수시로 변경하는데 많은 제약이 따르는데, 이를 보완하기 위해 만들어진 것이 스타일 시트이고 스타일 시트의 표준안이 바로 CSS이다. 간단히 스타일 시트라고도 한다.
